# IES Widget

Статистический виджет с иконкой и счётчиком пользователей.

## UX-сценарии

- Hover: лёгкая пульсация starburst-фона
- Click: (planned) раскрытие детальной статистики

## Данные

- Источник: static (число в mock)
- Статус: **mock**

## Experience

- Hover sound: `hoverSoft` (debounced)
- Haptic: `light` on tap

## Файлы

- `ies.styles.css` — позиционирование иконки
- `ies.animations.css` — пульсация фона
