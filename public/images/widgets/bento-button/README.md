# Bento button — иконки плиток

SVG-иконки для bento-плиток (канал, IES, YouTube). Лежат отдельно от разметки виджета.

| Файл | Плитка |
|------|--------|
| `bento-mychannel.svg` | Telegram-канал |
| `bento-ies.svg` | IES |
| `bento-youtube.svg` | YouTube |

В `BentoWidget.astro` иконки сейчас **inline** (пути SVG в DOM), эти файлы — исходники / резерв для замены.

Виджет: [`src/widgets/bento/README.md`](../../../src/widgets/bento/README.md).
