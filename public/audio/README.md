# UI Sounds

Положите сюда аудиофайлы:

- `tap.mp3` — клик по кнопкам и ссылкам
- `hover-soft.mp3` — hover на кейс (debounced)
- `page-transition.mp3` — переход между страницами

Пути заданы в `src/experience/feedback/sounds.config.ts`.

Если файлов нет, `AudioEngine` молча пропускает воспроизведение.
